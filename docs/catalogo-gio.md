# Catálogo Gio Handmade — estrutura pronta pra Canva

**Como usar:** abre o Canva → "Apresentação" → formato A4 vertical (ou "Lookbook"). Cria 1 página por bloco abaixo. Cola o texto. Solta a foto. Pronto.

**Paleta sugerida:** off-white (#F5F1EA) + preto (#1A1A1A) + dourado fosco (#B8935A). Tipografia: serifada elegante no título (ex: Playfair Display), sans-serif limpa no corpo (ex: Inter ou Montserrat).

- [ ] Salvar no celular dela em "Documentos" pra mandar rápido no WhatsApp

---

## Dicas de uso

1. **Manda como PDF**, não imagem solta. PDF abre direto no celular, parece profissional.
2. **Quando cliente chama:** "Posso te mandar nosso catálogo de modelos? Ali você vê os formatos e faixa de preço, aí a gente customiza junto." → manda PDF → muda o jogo.
3. **Atualiza a cada 3 meses** ou quando uma peça nova vira hit. Não precisa refazer tudo, só trocar modelos.
4. **Versão "stories"**: depois que tiver pronto, recorta cada peça em formato 9:16 e sobe nos destaques do Instagram.
